Typeless but Free —— 本地语音听写（Windows 免安装版）
=====================================================

按住热键说话，松开，文字自动出现在你光标处。
语音在你电脑本地转写（不上传），可选接 AI 润色去口水词。

【怎么用】
1. 双击 TypelessButFree.exe
2. 第一次会弹出设置窗：
   - 想用 AI 润色：填一个 OpenAI 兼容的 API Key（DeepSeek / OpenAI 等都行）
   - 不想用润色：Key 留空即可（纯本地，啥都不上传）
   - 其它默认就能用；中国大陆把"模型下载源"改成 https://hf-mirror.com 下载更快
3. 点"保存并启动"
4. 第一次启动会自动下载语音识别模型（几百 MB，下一次就不用了）
5. 按住 Alt+V 说话 → 松开 → 看右下角 → 文字自动插入

【说明】
- 这是 CPU 版，任何 Windows 电脑都能跑（不需要显卡），识别速度取决于你的 CPU。
- 有 NVIDIA 显卡想要更快？去 GitHub 跑源码版（README 里有说明），能快好几倍。
- 配置随时可改：把 exe 拖一下或重开会用旁边的 config.json；也可命令行 TypelessButFree.exe --settings 重开设置窗。
- 所有文件（exe / config.json / 下载的模型）都在这个文件夹里，删文件夹即彻底卸载。

【隐私】
语音只在本地转写，不上传。只有在你填了 Key 开启润色时，才会把"转写后的文字"发给你选的 AI 服务。
